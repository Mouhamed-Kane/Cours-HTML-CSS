Défilement vers le bas

Faites défiler vers le bas pour voir l'effet collant.

Barre de navigation collante

La barre de navigation coller au sommet lorsque vous atteignez sa position de défilement..

Note: Internet Explorer ne prend pas en charge le positionnement collant et Safari exige un préfixe -webkit-.
