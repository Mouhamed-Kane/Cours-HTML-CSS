Nav latéral fixe pleine hauteur

Essayez de faire défiler cette zone et voyez comment le sidenav se colle à la page.

Remarquez que cet élément div a une marge gauche de 25 %. Cela s'explique par le fait que la navigation latérale est définie sur une largeur de 25 %. Si vous supprimez la marge, la navigation latérale se superposera à cet élément div.

Remarquez également que nous avons défini overflow:auto pour sidenav. Cela ajoutera une barre de défilement lorsque le sidenav est trop long.
